# Flutter Local Auth

Um pequeno exemplo para utilização do package local_auth no Flutter, para autenticação biométrica

- Version: Flutter 3.10.4

### Run
1. Clone o repositório
2. Execute `flutter pub get`
3. Execute `flutter run`
